DeepWork 에이전트 요청 예시

  1. 이 페이지 링크를 DeepWork 에이전트에게 제공합니다.
  2. 에이전트가 브라우저로 페이지에 접속해 HWPX 서식을 다운로드합니다.
  3. 다운로드한 서식을 워크스페이스에 두고 HWPX 빈칸 채우기를 실행합니다.
  4. 채워진 HWPX 결과 파일을 생성하고, 빈칸이 실제 값으로 바뀌었는지 확인합니다.

중요한 조건:
- 내가 파일을 직접 첨부하거나 로컬 경로를 알려주는 것이 아니라, 반드시 위 웹페이지에 브라우저로 접속해서 HWPX 파일을 다운로드해줘.
- 다운로드한 파일 이름은 `hwpx-form-fill-example.hwpx`여야 해.
- 원본 파일은 덮어쓰지 말고, 빈칸을 채운 새 HWPX 파일을 별도로 저장해줘.
