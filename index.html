DeepWork v0.4.2 macOS public

전체 노트
  • Panel 실행이 실제 에이전트 도구 경로와 연결되어, 패널 멤버 작업과 실행 타임라인을 더 명확하게 확인할 수 있습니다.
  • Incident diagnostics와 support bundle이 확장되어 현재 스레드 진단, 타임라인 이벤트, 내보내기 흐름을 함께 수집할 수 있습니다.
  • Browser Connection / Chrome Bridge 경로를 정리하고 직접 도구 실행을 안정화했습니다.
  • 앱 버전 표시, 모델 선택 라벨, 로컬 사용량 예산 편집, readiness toast 동작을 다듬었습니다.
  • 공개 macOS 자산을 Apple Silicon arm64 설치 및 업데이트 경로로 명확히 정리했습니다.
  • 릴리스 UI에서 아직 준비 중인 Work overview entrypoint가 노출되지 않도록 정리했습니다.

승인 전 임시 ZIP 설치

  1. Chrome 확장 ZIP을 다운로드하고 압축을 풉니다.
  2. Chrome 주소창에 chrome://extensions를 엽니다.
  3. 오른쪽 위 개발자 모드를 켠 뒤 압축해제된 확장 프로그램을 로드합니다를 선택합니다.
  4. 압축을 푼 폴더를 선택하고, DeepWork 설정의 브라우저 연결에서 연결 활성화 또는 복구를 실행합니다.
  5. Chrome에서 확장 아이콘을 한 번 열고 DeepWork에서 연결 테스트를 실행합니다.
