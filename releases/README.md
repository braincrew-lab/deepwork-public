# Releases

This directory contains public release metadata, not installer binaries.

Official installer artifacts should be uploaded through GitHub Releases:

```text
https://github.com/braincrew-lab/deepwork-public/releases
```

Use metadata files in this directory to point public pages and update checkers
at reviewed release artifacts.

Do not commit DMG, ZIP, MSIX, AppX, EXE, or other large binary release
artifacts directly to git.
